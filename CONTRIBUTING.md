# Contributing to Post Ou Cancer

First off, thank you for considering contributing to Post Ou Cancer! It's people like you that make this such a great community. We welcome any type of contribution, not only code.

## Getting Started

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally: `git clone https://github.com/your-username/post-ou-cancer.git`
3. **Set up the development environment** by following the instructions in the README.md.
4. **Create a new branch** for your changes. Please use a descriptive branch name.

```bash
# Good branch names
git checkout -b feature/add-user-profile-page
git checkout -b fix/login-form-validation
```

## Making Changes

- **Code Style**: Please follow the existing code style. We use StyleCop for C# code to enforce conventions, so make sure your code passes the linter checks.
- **Tests**: Please add unit or integration tests for any new features or bug fixes. Our goal is to maintain a high test coverage (>80%).
- **Commit Messages**: Write clear and concise commit messages. We follow the Conventional Commits specification.

## Submitting a Pull Request

1. **Push your branch** to your fork on GitHub.
2. **Create a Pull Request** (PR) to the `main` branch of the `ahnwooseon/post-ou-cancer` repository.
3. **Fill out the PR template**, describing the changes you've made and linking to any relevant issues.
4. **Ensure all CI checks pass**. Your PR will be automatically built and tested by GitHub Actions.

## Code of Conduct

This project and everyone participating in it is governed by our Code of Conduct. By participating, you are expected to uphold this code.

## Reporting Bugs

If you find a bug, please open an issue on our GitHub Issues page. Please provide as much detail as possible, including steps to reproduce the bug.

Thank you for your contribution!
